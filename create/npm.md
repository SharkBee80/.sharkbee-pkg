# pnpm

## 通过 npm 全局安装

```bash
npm install -g pnpm
```

## pnpm 镜像

```bash
pnpm set registry https://registry.npmmirror.com
```

1. https://registry.npmjs.org/
2. https://registry.npmmirror.com/

## 升级版本

```bash
pnpm add -g pnpm to update
```
